== Description ==

**WPC Pexels** helps you search millions of free photos from pexels.com then insert into content or set as featured image very quickly.

= Features =

- Search millions of free photos from Pexels
- Quick insert image with original URL
- Upload image(s) to Media Library (Premium Version)
- Use image as featured image
- Multiple select support
- WPML integration

== Installation ==

1. Unzip the wp-pexels.zip
2. Copy wp-pexels folder to wp-content/plugins
3. Go to Plugins > Installed Plugins, find WPC Pexels and click Active
4. Now when you create/edit the post, you can see the Pexels button above the editor
5. Enjoy!
